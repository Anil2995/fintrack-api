Finance Dashboard API

Interactive Live Demo

Select a role below to authenticate via the Live API. Watch the network requests in the console below.

Administrator

admin@finance.dev

  • Full CRUD on all records
  • Access aggregated data
  • Manage user roles & status
Analyst

analyst@finance.dev

  • Read & Write records
  • Access trend analysis
  • Cannot delete records
  • Cannot manage users
Viewer

viewer@finance.dev

  • View summary dashboard
  • Read own records
  • Cannot create/edit records
  • No advanced analytics

API Reference Guide

Endpoints available in this implementation, protected by RBAC middleware.

Authentication & Users
POST /api/auth/register
POST /api/auth/login
GET /api/users (Admin ONLY)
Financial Records & Analytics
GET /api/records?search=&type=
POST /api/records (Admin/Analyst)
DEL /api/records/:id (Admin ONLY)
GET /api/dashboard/summary
